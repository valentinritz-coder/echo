# Principes de confidentialité (MVP)

- Les données sont stockées localement par défaut (fichiers + futur SQLite).
- L'utilisateur peut supprimer ses données locales à tout moment.
- Aucune fonctionnalité de "chat avec les morts" n'est envisagée.
- Les enregistrements audio sont considérés comme des données privées sensibles.
- Toute future intégration IA devra être explicitement signalée.
- Le partage ou l'export ne sera jamais activé silencieusement.
- Le produit doit privilégier la sobriété et le consentement.
