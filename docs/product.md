# Echo — Concept produit (MVP)

Echo est une capsule de vie centrée sur la mémoire personnelle.
L'utilisateur enregistre des réponses audio à des questions guidées.
Les audios sont stockés localement dans un premier temps.
Une transcription IA est prévue dans une prochaine itération.
Le contenu sera organisé dans une bibliothèque consultable.
L'objectif est de conserver des souvenirs authentiques sur la durée.
Le MVP commence par une base simple: API + interface légère.
La collecte et la consultation seront progressives et explicables.
La stack doit rester locale, lisible, et extensible.
La priorité actuelle: robustesse du socle technique.
